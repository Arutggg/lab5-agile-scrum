# Проект: Task Manager - Планировщик задач

## Описание проекта
Приложение для управления задачами с интуитивным интерфейсом. 
Позволяет создавать, редактировать и удалять задачи. 
Поддерживает категории задач и напоминания. 
Работает на всех устройствах. 
Цель - помочь пользователям организовывать своё время.
